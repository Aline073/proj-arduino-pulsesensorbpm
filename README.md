Este projeto propõe a construção de um sistema portátil de monitoramento de sinais vitais — frequência cardíaca (BPM) — utilizando o sensor de pulso, um microcontrolador Arduino com chip Wi-Fi, e um buzzer para alertas. A condição é criar uma ferramenta acessível, pode ser muito útil para controle de rotina de exercícios, atividade físicas diárias ou mesmo para fins didáticos e a prevenção de problemas cardiovasculares.
